BEARMAPS

April 2017, CS 61B

Using Java, I implemented the backend of a web mapping application that allows users to move around and zoom in to a map of Berkeley. Users can click two points on the map to find the shortest route between the points. I also added an autocomplete search feature.

WHAT I LEARNED

I used multiple data structures and graph algorithms in creating the end result. I also learned how to make my programs more time and space efficent.
